# Automated Log Monitoring and Alert System

## Overview
This project is a Python-based automated log monitoring system. It reads log data and detects suspicious activity such as repeated failed login attempts.

## Project Objective
The goal of this project is to demonstrate how security automation can reduce manual log review and improve threat detection.

## Features
- Reads log files
- Detects repeated failed login attempts
- Generates alert messages
- Demonstrates basic security automation using Python

## Technologies Used
- Python 3
- GitHub
- Basic pattern matching

## How to Run
1. Download or clone this repository.
2. Make sure Python 3 is installed.
3. Open the project folder.
4. Run:

```bash
python log_monitor.py
